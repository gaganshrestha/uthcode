Explaination
============

In this program we are going to convert a string of hexadecimal digits into Integer value.
If give input as F then the output should be 15.  This is done by the htoi function::

	int htoi(char s[])
    {
    int hexdigit,i,inhex,n;
    i = 0;
    if( s[i] == '0')
    {
        ++i;
        if(s[i] == 'x' || s[i] == 'X')
            ++i;
    }
    
    n = 0;
    inhex = YES;

    for(;inhex==YES;++i)
    {
        if(s[i] >='0' && s[i] <='9')
            hexdigit= s[i] - '0';
        else if(s[i] >='a' && s[i] <='f')
            hexdigit= s[i] -'a' + 10;
        else if(s[i] >='A' && s[i] <='F')
            hexdigit= s[i] -'A' + 10;
        else
            inhex = NO;
        
        if(inhex == YES)
            n = 16 * n + hexdigit;
    }
    return n;
    }
In the above fragment of the program we declare some variables such as hexdigit,i,inhex,n.  
In the For loop assign inhex to YES and increment the I.  Then we check each character 0 to 9 and a to f.  Then we check the input and multiply each character by the base of 16 and give the output
